Screencam demos are mini-videos that show you how to do things with NXTCamView.

We have screencams on -

  1. Setting up COMPorts (300k) - how to configure the COMPort to connect to the NXTCam
  2. Capture and Select (4meg) - how to connect to the NXTCam, capture an image, and select colors to track
  3. Testing Tracking - how to test the tracking (Sorry - this is not ready yet...)
  4. Sample Capture (48meg) - sample run of a capture, color selection and tracking. NOTE: This is a BIG file (48Meg) and may take a while to download!
